## 二、模块详细设计

### 2.1 线路侧光接口模块

**协议依据**: G.957 (光接口), G.703 (电接口可选)

| 接口信号 | 方向 | 说明 |
|---------|------|------|
| `RX_P/N` | 输入 | 光接收差分信号，STM-1 速率 155.52 Mbit/s |
| `TX_P/N` | 输出 | 光发送差分信号 |
| `RX_LOS` | 输出 | 光信号丢失指示 |

**数据流**:
- **接收方向**: 光信号 → 光电转换 → 时钟恢复(CDR) → 串并转换(8/16位并行) → 155.52MHz 字节流 → **帧定位模块**
- **发送方向**: **帧组装模块** → 并串转换 → 电光转换 → 光信号

---

### 2.2 STM-1 段开销处理模块 (G.783 §10-11)

**协议依据**: G.707 §9 (SOH定义), G.783 §10 (RSOH), §11 (MSOH)

#### 2.2.1 再生段开销处理 (RSOH)

| 子模块 | 功能 | 协议章节 |
|--------|------|---------|
| 帧定位 | A1/A2 字节检测 (F6 28), 帧失步告警(OOF/LOF) | G.707 §9.2.2.1 |
| 扰码/解扰码 | X^7+X^6+1 自同步扰码 | G.707 §6.5 |
| B1 监测 | 再生段 BIP-8 误码监测 | G.707 §9.2.2.4 |
| DCC-R | D1-D3 字节提取/插入 (192kbit/s 管理通道) | G.707 §9.2.2.7 |
| E1 公务 | 再生段公务字节提取/插入 | G.707 §9.2.2.5 |

**数据流**:
```
STM-1 串行流 → [A1/A2 检测] → [解扰码] → [B1 计算/校验] 
    → 分离 RSOH(前3行) 与 MSOH+净荷(后6行)
    → RSOH 终端处理 (J0/B1/DCC/E1)
    → 输出 MSOH+净荷 到 MSOH 处理单元
```

#### 2.2.2 复用段开销处理 (MSOH)

| 子模块 | 功能 | 协议章节 |
|--------|------|---------|
| B2 监测 | 复用段 BIP-24 误码监测 | G.707 §9.2.2.10 |
| K1/K2 APS | 自动保护倒换字节处理 | G.841, G.707 §9.2.2.11 |
| S1 SSM | 同步状态消息提取/插入 | G.707 §9.2.2.13 |
| M1 REI | 远端误码指示 | G.707 §9.2.2.14 |
| DCC-M | D4-D12 字节提取/插入 (576kbit/s) | G.707 §9.2.2.8 |

**数据流**:
```
输入(MSOH+净荷) → [B2 计算/校验] → [K1/K2 APS 处理] → [分离 MSOH 与净荷]
    → MSOH 终端处理 (B2/K1/K2/S1/M1/DCC-M)
    → 输出 AU-4 指针区域 + VC-4 净荷 到 AU 处理单元
```

### 2.3 AU-4 指针处理模块 (G.707 §8.1, G.783 §11.3.1)

**协议依据**: G.707 第 8.1 节 (AU-4 指针), G.783 第 11.3.1 节 (MSn/Sn_A 适配)

| 接口信号 | 方向 | 说明 |
|---------|------|------|
| `au4_in_data[15:0]` | 输入 | 16 位并行数据 (9.72MHz 时钟) |
| `au4_ptr_h1h2[15:0]` | 输入 | H1/H2 指针字节值 |
| `au4_ptr_h3[7:0]` | 输入 | H3 调整字节 |
| `vc4_out_data[31:0]` | 输出 | 32 位并行 VC-4 数据 |
| `vc4_frame_start` | 输出 | VC-4 帧起始指示 |
| `ptr_inc/dec` | 输出 | 指针调整事件指示 |

**功能块**:

1. **指针解释器** (G.707 §8.1.6)
   - H1/H2 解析：N-bits[15:12]="0110"(正常)/"1001"(NDF)
   - I-bits/D-bits 5 位多数决检测正负调整
   - 指针值范围 0-782 (10 位)
   - 连续 3 次接收新值才确认 (G.707 §8.1.6 规则 2)

2. **帧相位恢复**
   - 根据指针值定位 VC-4 起始位置 (J1 字节)
   - 处理正调整 (I-bits 反转，H3 后填充 3 字节)
   - 处理负调整 (D-bits 反转，H3 承载 3 字节数据)
   - NDF 处理 (新数据标志 "1001")

**数据流**:
```
AU-4 帧流 → [H1/H2/H3 提取] → [指针状态机解析] 
    → [VC-4 起始位置计算] → [正负调整处理] 
    → 连续 VC-4 字节流 → [写入 VC-4 缓冲器]
```

### 2.4 高阶通道连接 (HPC) - VC-4 交叉 (G.783 §9.4)

**协议依据**: G.783 第 9.4 节 (HPC 功能), G.707 第 7.3 节 (VC-4 结构)

| 参数 | 值 | 说明 |
|------|-----|------|
| VC-4 帧长 | 2349 字节 | 9 行 × 261 列 |
| VC-4 速率 | 150.336 Mbit/s | 9 × 261 × 8000 × 8 |
| 交叉粒度 | VC-4 (150M) | 支持 E4 直接连接 |

**数据流**:
```
┌─────────────────────────────────────────┐
│           VC-4 交叉连接矩阵              │
│                                         │
│  输入 VC-4 #0 ──┐                       │
│  输入 VC-4 #1 ──┼──► [交叉开关阵列] ──► 输出 VC-4 #0
│  输入 VC-4 #2 ──┤         │           输出 VC-4 #1
│      ...        │         │              ...
│  输入 VC-4 #N ──┘         ▼           输出 VC-4 #N
│                    连接配置表 (SEMF)
└─────────────────────────────────────────┘
```

**控制流**:
- SEMF (同步设备管理功能) 通过 **连接配置表** 控制交叉矩阵
- 支持 **单向/双向** 连接
- 支持 **广播** (一个输入到多个输出)
- 支持 **环回** (本地/远端)

### 2.5 VC-4 解复用与 TUG3 分离 (G.707 §7.3.7)

**协议依据**: G.707 第 7.3.7 节 (TU-3 编号), 第 7.3.3 节 (TUG3 结构)

VC-4 净荷区 (列 10-270) 包含 3 个 TUG3:
- **TUG3 #1**: 列 4,7,10...259 (VC-3 或 TUG2 复用)
- **TUG3 #2**: 列 5,8,11...260
- **TUG3 #3**: 列 6,9,12...261

**数据流** (接收方向):
```
VC-4 净荷流 → [列分离器] 
    ├──► TUG3 #1 流 ──► [类型判别: TU3 或 TUG2?]
    ├──► TUG3 #2 流 ──► [类型判别]
    └──► TUG3 #3 流 ──► [类型判别]
    
类型判别依据: C2 字节 (信号标签)
    - C2 = 0x04: TU3 结构 (E3 业务)
    - C2 = 0x12: TUG2 结构 (E1 业务)
```

### 2.7 VC-12 处理分支 (G.707 §8.3, G.783 §12.3)

**协议依据**: G.707 第 8.3 节 (TU-12 指针), 第 7.3.9 节 (TU-12 编号), G.783 第 12.3.1.2 节

#### 2.7.1 TUG2 解复用

VC-4 经 TUG3 后，若配置为 E1 业务，继续分解:
- 每个 TUG3 含 **7 个 TUG2**
- 每个 TUG2 含 **3 个 TU-12**
- 总计: 3 × 7 × 3 = **63 个 TU-12** (E1)

**数据流**:
```
TUG3 流 → [TUG2 分离器 (7 路)] 
    → TUG2 #0 ──► [TU-12 #0,#1,#2]
    → TUG2 #1 ──► [TU-12 #3,#4,#5]
    → ...
    → TUG2 #6 ──► [TU-12 #18,#19,#20]
```

#### 2.7.2 TU-12 指针处理

| 参数 | 值 | 协议依据 |
|------|-----|---------|
| 指针位置 | V1/V2 字节 (每 4 帧复帧) | G.707 §8.3.1 |
| 指针范围 | 0-139 | G.707 §8.3.2 |
| 调整字节 | V3 (负调整机会) | G.707 §8.3.3 |
| 复帧指示 | H4 字节 (G.707 §8.3.8) | 4 帧复帧 (500μs) |

**数据流**:
```
TU-12 流 → [H4 复帧定位] → [V1/V2/V3/V4 提取]
    → [500μs 复帧同步] → [TU-12 指针解释]
    → [VC-12 起始定位] → 连续 VC-12 流 (36 字节/帧)
```

| VC-12 帧长 | 36 字节 | 9 行 × 4 列 |
| 速率 | 2.304 Mbit/s | 63 × E1 |
| 复用结构 | VC-12 → TU-12 → TUG2 → TUG3 → VC-4 | G.707 图 7-2 |

**数据流**:
```
63 路 VC-12 流 ──► [VC-12 交叉矩阵 (63×63)] ──► 输出 VC-12
                                                    │
                                                    ▼
                                            [C-12 去映射]
                                                    │
                                                    ▼
                                            E1 (2.048M) 接口
```

#### 2.7.4 C-12 容器与 E1 适配 (G.707 §10, G.703 §8)

**协议依据**: G.707 第 10.1 节 (异步映射), G.703 第 8 节 (2048kbit/s 接口)

- **异步映射**: 比特同步映射 (E1 时钟与 SDH 不同步)
- **码型**: HDB3 (AMI 禁用)
- **帧结构**: PCM30/32 帧 (256 比特，125μs)
- **开销**: 
  - V5 字节 (误码监测、信号标记、RDI)
  - J2 字节 (通道踪迹)
  - N2 字节 (串联连接监视)
  - K4 字节 (保护倒换)
