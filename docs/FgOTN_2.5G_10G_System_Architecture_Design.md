## 二、同步低速信号映射模块设计

### 2.1 协议依据与映射关系

| 客户信号 | 目标 ODU | 映射方式 | 协议章节 |
|---------|---------|---------|---------|
| STM-1 (155M) | ODU0 | GMP (通用映射) | G.709 §17.7 |
| STM-4 (622M) | ODU1 | AMP/BMP | G.709 §17.2 |
| STM-16 (2.5G) | ODU2 | AMP/BMP | G.709 §17.2 |
| E3 (34M) | ODU0 | GMP | G.709 §17.7 |
| E4 (140M) | ODU0 | GMP | G.709 §17.7 |
| SyncE (1G/10G) | ODU0e/ODU2e | 同步映射 | G.709 §17.8 |
| 1588 PTP | ODUflex | GFP-F 映射 | G.709 §17.4 |
| 专用时钟 | ODUflex | 专用映射 | G.8261 |
