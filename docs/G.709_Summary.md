# G.709 主要内容梳理

> 来源：ITU-T G.709/Y.1331 (2020) Amendment 1 (Edition 6.1)
> 文档：ref/G.709_202012.pdf（共290页）

---

## 1. 标准概述

**G.709/Y.1331** 定义了光传送网络（OTN）的接口规范，涵盖：
- OTN 层次结构与复用映射
- 开销功能（支持多波长光网络）
- 帧结构与比特率
- 客户信号映射格式

**版本演进**：本版为 Edition 6.1（2020年12月），主要增加了 OTUCn、OTU25/50、GCC 比特率表、OTUCn-M 行为描述等内容。

---

## 2. OTN 分层结构

OTN 采用 **光层 + 电层（数字层）** 的两层结构：

### 光层（Optical Layer）

| 层次 | 全称 | 功能 |
|------|------|------|
| **OTS** | 光传输段 | 光放大器之间的传输 |
| **OMS** | 光复用段 | 光复用/解复用点之间 |
| **OCh** | 光通道 | 端到端光通道 |
| **OTSi** | 光支路信号 | 承载 OTU 的单个光信号 |
| **OTSiG** | 光支路信号组 | 支持一个 OTU 的 OTSi 集合 |
| **OTSiA** | 光支路信号组件 | OTSiG + 非关联开销(OTSiG-O) |

### 数字层（Digital Layer）

| 层次 | 功能 | 主要类型 |
|------|------|----------|
| **OTU** | 传送单元，提供段监控 | OTUk(k=1,2,3,4)、OTU25、OTU50、OTUCn |
| **ODU** | 数据单元，提供路径/串联监控 | ODUk(k=0,1,2,2e,3,4,flex)、ODUCn |
| **OPU** | 净荷单元，适配客户信号 | OPUk、OPUCn |

---

## 3. 帧结构

### OTUk 帧（k=1,2,3,4）
- **4080列 × 4行**
- 含 **256列 FEC** 区域
- 帧周期约 12.191 μs（OTU1）

### OTUCn/OTU25/OTU50 帧
- **3824列 × 4行**
- **无内置 FEC**（FEC 由接口特定规范定义）
- OTUCn 由 n 个交错帧组成（每帧一个 OTUC 实例）

### ODU 帧结构
```
行1: FAS(6B) + MFAS(1B) + OTU-OH(7B) + ODU-OH(34B) + OPU-OH(2B) + 净荷区
行2-4: ODU-OH(不同位置) + OPU净荷
```

### OPU 帧结构
- **4行 × 3810列**
- 第15-16列：OPU开销（PSI、JC等）
- 第17-3824列：OPU净荷（4×3808字节）

## 4. 开销体系

### OTU 开销

| 开销 | 说明 |
|------|------|
| **SM** | 段监控：TTI、BIP-8、BDI、BEI/BIAE、IAE、STAT |
| **GCC0** | 通用通信通道（~13.768 Mbit/s） |
| **OSMC** | OTN同步消息通道（SSM/eSSM/PTP） |

### ODU 开销

| 开销 | 说明 |
|------|------|
| **PM** | 路径监控：TTI、BIP-8、BDI、BEI、STAT、DMp（时延测量） |
| **TCM1-6** | 6级串联连接监控，支持嵌套/级联配置 |
| **GCC1/2** | 两个通用通信通道（各~13.768 Mbit/s） |
| **APS/PCC** | 自动保护倒换/保护通信通道 |

### OPU 开销

| 开销 | 说明 |
|------|------|
| **PSI** | 净荷结构标识符（PT=净荷类型，CSF=客户信号失效） |
| **JC** | 调整控制（用于 AMP/GMP 映射） |
| **OMFI** | OPU多帧指示器（支持20/40/80帧多帧） |

### 光层开销（非关联开销）

- **OTS-O**: TTI、BDI-P、BDI-O、PMI
- **OMS-O**: FDI-P、FDI-O、BDI-P、BDI-O、PMI、MSI
- **OCh-O/OTSiG-O**: FDI-P、FDI-O、OCI、TTI、TSI、BDI-P、BDI-O

## 6. 复用映射结构

```
客户信号
   ↓
OPUk/OPUflex（映射）
   ↓
ODUk/ODUflex
   ↓
   ├──→ OTUk/OTUCn（直接承载）
   ↓
   └──→ ODTUjk/ODTUk.ts → ODTUG → OPUk/OPUCn（复用）
            ↓
         ODUk/ODUCn
            ↓
         OTUk/OTUCn
```

**ODU 复用**：支持最多 10n 个 ODUk(k=0,1,2,2e,3,4,flex) 复用到一个 OPUCn。

## 7. 客户信号映射方式

| 映射方式 | 英文缩写 | 适用场景 |
|---------|---------|---------|
| **异步映射** | AMP | CBR2G5、CBR10G 等，±20 ppm |
| **比特同步映射** | BMP | 时钟同步的 CBR 信号 |
| **通用映射** | GMP | 各种速率 CBR 信号，±100 ppm |
| **空闲映射** | IMP | 64B/66B 编码信号（FlexE、以太网） |
| **GFP-F 映射** | GFP | 分组客户信号（以太网、MPLS 等） |

### 支持的主要客户信号

- **SDH**: STM-1/4/16/64/256
- **以太网**: 1000BASE-X、10GBASE-R、40GBASE-R、100GBASE-R、200GbE、400GbE
- **光纤通道**: FC-100/200/400/800/1600/3200
- **InfiniBand**: SDR/DDR/QDR
- **CPRI**（通用公共无线接口）
- **SDI**（串行数字接口）
- **FlexE 信号**

## 8. 维护信号

| 信号 | 说明 |
|------|------|
| **AIS** | 告警指示信号，向下游指示信号失效 |
| **OCI** | 开放连接指示，指示矩阵连接已打开 |
| **LCK** | 锁定，指示连接被管理锁定 |
| **CSF** | 客户信号失效，指示客户信号丢失 |

---

## 9. 关键特性

1. **前向纠错（FEC）**：OTUk 使用 RS(255,239) 编码，OTUCn/OTU25/OTU50 的 FEC 由接口规范定义

2. **串联连接监控（TCM）**：支持6级嵌套/级联的 TCM，用于多运营商网络监控

3. **时延测量**：ODU PM 和 TCM 支持双向时延测量（DMp/DMti）

4. **FlexE 支持**：支持 FlexE Client 和 FlexE Aware 两种映射模式

5. **同步消息传输**：OSMC 支持 SSM、eSSM 和 PTP 消息传输
