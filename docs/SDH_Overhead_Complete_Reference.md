### 4.2 VC-12/VC-11 POH (V5 字节详细结构)

```
VC-12/VC-11 POH - V5 字节结构 (G.707 Figure 9-9)
═══════════════════════════════════════════════════════════════════════════════

V5 字节位于 VC-12/VC-11 第1列第1行，提供以下功能:

    ┌─────────────────────────────────────────────────────────────────┐
    │  b1  │  b2  │  b3  │  b4  │  b5  │  b6  │  b7  │  b8  │
    ├──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┤
    │ BIP-2  │  REI │  RFI │    信号标签    │  RDI │
    │ 误码监测│ 远端误码│ 远端故障│  (3比特)     │ 远端缺陷│
    └─────────────────────────────────────────────────────────────────┘

功能详解:
- b1-b2: BIP-2 - 对整个 VC-12/VC-11 前一帧进行 BIP-2 偶校验
- b3:    LP-REI (远端误码指示) - 回告 BIP-2 检测到的误码
- b4:    RFI (远端故障指示) - 保留为将来的远端故障指示
- b5-b7: 信号标签 - 指示 VC-12/VC-11 装载内容:
         000=未装载, 001=已装载非特定净荷, 010=异步映射, 
         011=比特同步映射, 100=字节同步映射, 101=扩展信号标签,
         110=OTUk, 111=保留
- b8:    LP-RDI (远端缺陷指示) - 向发送端指示接收端检测到缺陷

═══════════════════════════════════════════════════════════════════════════════
```

## 五、指针结构一览 (G.707 §8)

### 5.1 AU-4 指针 (H1, H2, H3)

```
AU-4/TU-3 指针编码 (H1, H2 字节结构)
═══════════════════════════════════════════════════════════════════════════════

    ┌─────────────────────────────────────────────────────────────────────┐
    │  N  │  N  │  N  │  N  │  S  │  S  │  I  │  D  │  I  │  D  │ ... │  I  │  D  │
    │ b1  │ b2  │ b3  │ b4  │ b5  │ b6  │ b7  │ b8  │ b9  │ b10 │     │ b15 │ b16│
    └─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┘
     
     N-bits (b1-b4): 新数据标志 (NDF)
                     0110 = 正常操作
                     1001 = 新数据标志 (允许指针任意变化)
     
     SS-bits (b5-b6):  AU类型指示 (已废弃，不再使用)
     
     I-bits (b7,b9,b11,b13,b15): 正调整比特 (5位多数表决)
     D-bits (b8,b10,b12,b14,b16): 负调整比特 (5位多数表决)
     
     指针值: b7-b16 的 10 位二进制值 (0-782 对于 AU-4, 0-764 对于 TU-3)

操作规则:
- I-bits 多数反转 (+): 正调整，VC 速率慢，H3 后为填充字节
- D-bits 多数反转 (-): 负调整，VC 速率快，H3 携带 VC 数据
- NDF=1001: 新数据，忽略 I/D 变化，直接使用新指针值

═══════════════════════════════════════════════════════════════════════════════
```
